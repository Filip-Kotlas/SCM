◆ read_binMatrix()

void read_binMatrix ( const std::string & file,
std::vector< int > & cnt,
std::vector< int > & col,
std::vector< double > & ele )

A sparse matrix in CRS format (counter, column index, value) is read from a binary file. 

   The binary file has to store 4 Byte integers and 8 Byte doubles and contains the following data:
   - (int*4) Number of rows
   - (int*4) Number of non-zero elements/blocks
   - (int*4) Number of non-zero matrix elements (= previous number * dofs per block)
   - (int*4) [#elements per row]
   - (int*4) [column indices]
   - (real*8)[matrix elements]
Parameters
[in]filename of binary file
[out]cntnumber of non-zero elements per row
[out]colcolumn indices of non-zero elements, C-Style
[out]elenon-zero elements of matrix

const std::vector< int > & cnt,
const std::vector< int > & col,
const std::vector< double > & ele )

A sparse matrix in CRS format (counter, column index, value) is written to a binary file. The memory is allocated dynamically.

The binary file has to store 4 Byte integers and 8 Byte doubles and contains the following data:

  • (int*4) Number of rows
  • (int*4) Number of non-zero elements/blocks
  • (int*4) Number of non-zero matrix elements (= previous number * dofs per block)
  • (int*4) [# elements per row]
  • (int*4) [column indices]
  • (real*8)[matrix elements]
Parameters
[in]filename of binary file
[in]cntnumber of non-zero elements per row
[in]colcolumn indices of non-zero elements, C-Style
[in]elenon-zero elements of matrix

Generates the mesh hierachy with nlevel meshes starting from coarse mesh cmesg .

The refined meshes are generated by edge bisection. All memory is allocated but stiffness matrices are yet not calculated

Parameters
[in]cmeshinitial coarse mesh
[in]nlevelnumber of meshes in hierarchy, including the initial coarse mesh
